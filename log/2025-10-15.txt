[使用者]
C 語言想做類似的東西，但是成果是印出到 A4 紙上，是可行的嘛？

[助手]
完全可行！有幾種方案：

**方案一：輸出 SVG 格式**（推薦）
- C 生成 SVG 文字檔，瀏覽器打開就能列印
- SVG 是向量圖，印出來很清晰
- 可精確控制 A4 尺寸（210×297mm）

**方案二：輸出 PostScript/PDF**
- 使用 `libharu` 或 `cairo` 函式庫生成 PDF
- 直接送印表機或存檔列印

**方案三：純文字 ASCII**
- 最簡單，用 `█` `░` 等字元
- 適合點陣印表機或轉成等寬字型 PDF

**方案四：輸出圖片**
- 用 `libpng` 或 `stb_image` 生成 PNG
- 高解析度適合列印

你想用哪種方式？我推薦 SVG，因為程式碼簡單、不需外部函式庫、列印品質又好。要不要我幫你寫一個 C 程式生成可列印的迷宮 SVG？

**使用說明：**
```bash
./maze              # 預設 31x41
./maze 51 71        # 自訂尺寸
./maze 21 31 my.svg # 自訂檔名
```

**特點：**
- SVG 格式適合列印到 A4 紙張（210×297mm）
- 自動計算格子大小並置中
- 綠色起點標示「起」，紅色終點標示「終」
- 向量格式，放大不失真
- 瀏覽器打開後直接 Ctrl+P 列印即可
